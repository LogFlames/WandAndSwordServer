Wand & Sword Website

Wand & Sword is made by Mr Ikea, and LogFlames. We are two young game developers. We are both experienced in coding with networks. LogFlames stands for all the Python code whilst Mr Ikea stands for all the GMS2 coding and Website. Mr Ikea also manages the servers and such things alike. We both own equal parts of the game and all decisions are decided by both of us.

And that is why our first game is a multiplayer one. Check the downloads page if you want to download the game.

If you would like to get your own game server for Wand & Sword's that will run 24/7. Donate on our patreon and then contact an admin on our discord server.

Game modes; There are 2 game modes in Wand & Sword, ranked is a 1v1 mode, you battle each other to the death until one of you reach 3 wins. The second game mode is a open world game mode were you can gather xp from slaying bosses with friends or going at it solo. You can trade items with other. Players can interact with each other via text chat.
